Приклад анімації

Засоби керування презентацією

Основні клавіші керування:

  • F5 - початок показу з першого слайда
  • Shift + F5 - початок з поточного слайда
  • Esc - вимкнути перо/лазерний вказівник

Під час показу презентації ви можете:

  • Використовувати мишу як лазерний вказівник
  • Змінювати колір вказівника (за замовчуванням червоний)
  • Переходити між слайдами
  • Збільшувати частини слайда

Нотатки доповідача

Нотатки доповідача - це поле під кожним слайдом, яке допомагає під час презентації.

Особливості нотаток:

  • Відображаються лише на вашому моніторі
  • Глядачі їх не бачать
  • Можна використовувати як сценарій
  • Допомагають згадати важливі моменти

Порада: Додавайте нотатки під час створення презентації, щоб не забути важливі деталі під час виступу.

Режим доповідача

Режим доповідача дозволяє:

  • Переглядати презентацію з нотатками на одному комп'ютері
  • Показувати лише слайди на екрані для аудиторії
  • Використовувати ескізи для вибору слайдів
  • Показувати слайди в довільному порядку

Додаткові можливості:

Зміна яскравості екрана, використання інструментів для малювання, збільшення фрагментів слайдів

Репетиція та записування часу

PowerPoint може записувати час відображення кожного слайда під час репетиції.

Функції записування:

  • Запис голосу та відео
  • Фіксація рухів рукописного вводу
  • Автоматичний хронометраж слайдів
  • Можливість повторного запису

Хронометраж можна використати для автоматичного показу слайдів без участі доповідача.

Що таке анімація?

Анімація

це динамічний візуальний ефект, доданий до текстового або графічного об'єкта слайда

Ефекти анімації дозволяють:

  • Оформити появу об'єктів
  • Налаштувати зникнення елементів
  • Створити переміщення об'єктів
  • Змінювати розмір та колір

Типи анімації

Анімацію можна додавати до:

  • Окремих об'єктів на слайді
  • Слайда в цілому
  • Групи слайдів

Алгоритм додавання анімації:

  1. Перейти до тексту або об'єкта
  2. Вибрати вкладку "Анімації"
  3. Обрати ефект анімації
  4. Налаштувати параметри ефектів

Управління анімацією

Анімацію можна запускати різними способами:

  • По клацанню - запуск за клацанням миші
  • Після попереднього - відразу після попереднього ефекту
  • З попереднім - одночасно з попереднім ефектом

Важливо: До одного об'єкта можна додати кілька анімацій різних типів!

Параметр "Час" дозволяє налаштувати тривалість та кількість повторень ефекту.

Рух об'єктів в презентації

Анімаційні ефекти групи "Шляхи переміщення" налаштовують переміщення об'єктів:

Приклад руху по траєкторії

Алгоритм додавання шляху:

  1. Клацнути об'єкт для анімації
  2. Вкладка "Анімація" → "Додати анімацію"
  3. Розділ "Шляхи переміщення"
  4. Вибрати потрібний варіант або "Користувацький шлях"

Групування об'єктів

Групування

це об'єднання об'єктів у одну групу для роботи з усією групою як з одним цілим

Алгоритм групування:

  1. Вибрати об'єкти, утримуючи клавішу Ctrl
  2. Викликати контекстне меню
  3. Обрати команду "Групувати"

Розгрупування: вибрати згрупований об'єкт → контекстне меню → "Розгрупувати"
